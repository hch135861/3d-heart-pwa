# 3D 爱心 PWA

完全离线的 3D 粒子爱心网页应用，适用于 iPhone、iPad 和现代桌面浏览器。

## iPhone 安装

1. 使用 Safari 打开 GitHub Pages 的 HTTPS 地址。
2. 等待爱心完整显示。
3. 点击 Safari 的“共享”按钮。
4. 选择“添加到主屏幕”，再点击“添加”。
5. 从主屏幕启动应用。

首次完整加载后，可开启飞行模式验证离线启动。若发布了新版本，联网打开一次页面即可获取更新。

## 本地验证

```powershell
python -m http.server 4173
```

然后访问 `http://127.0.0.1:4173/`。
